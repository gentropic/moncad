A small 2D CAD instrument — a precision board you operate by command and read by panel. Draw and edit with snapped clicks or typed coordinates; everything round-trips to DXF. Press F1 or ? any time to open this; Esc to close.

Getting around

  • Pan — drag with the middle mouse button.
  • Zoom — mouse wheel (zooms toward the cursor). = / - zoom in/out.
  • Zoom to fit — E (zoom extents).
  • The footer reads the live cursor position in world coordinates, the snap, the active layer, and the scale.

How you give commands

Every tool lives in one registry, reachable five ways — use whichever suits you:

  • Tool palette (left) — click a tool.
  • Menubar (top) — File / Edit / View / Help.
  • Command line (bottom) — type a command name (line), an alias (pl), or a coordinate.
  • Command palette — Ctrl+P, fuzzy search (matches names and aliases).
  • Right-click — a context menu of what's relevant (selection verbs, or board verbs); right-click also finishes the current tool.

Drawing

L LineChains segments — click a start, then each click commits a segment. Enter/Esc ends. (Each segment is its own entity.)
P PolylineOne connected entity. While drawing, type a for arc mode (spans curve tangent to the previous one) and l back to line; c closes, u undoes a vertex.
A ArcThree points: start, a point on the arc, end.
C CircleCentre, then a radius point — or type the radius.
PointPlaces node markers.
TextPlace a point, then type the text (type a number first to set the height).
I InsertPlace instances of a block (see Blocks).

Precision input

While a draw tool is active, type a coordinate into the command line instead of clicking:

x,yAbsolute world coordinate (matches the footer readout).
@dx,dyRelative to the last point.
d<a · @d<aPolar — distance d at angle a° (absolute / relative).

Snapping & ortho

  • Running snaps — end / mid / centre / node, shown as chips in the footer; click a chip to toggle it. F3 toggles snapping overall.
  • One-shot override — type a snap word (cen, end, mid, nod, non) to force just the next pick.
  • Cycle — Tab steps through candidates when several coincide (a ⇥ hint shows). Aperture: [ / ].
  • Grid snap — the GRID chip (object snaps still win). G toggles the reference grid.
  • Ortho — F8 locks the cursor to horizontal/vertical from the last point (snaps still win).

Selecting & editing

Click an entity to select it; drag a box to select many; with nothing active, hovering outlines what a click would pick. Block instances select as one unit. The affine tools need a selection first; the click-on-geometry tools act on whatever you click.

M Move · ⇧C Copy · R RotatePick a base/pivot, then a destination point or a typed value (angle in degrees).
Mirror · ScaleMirror across an axis (two points); Scale by a base point + a typed factor.
Array (grid / polar)Grid: columns × rows × spacing. Polar: a centre, a count, a fill angle.
T Trim · X ExtendClick the part to remove / an end to lengthen. The hover preview shows it (red = removed, green = added).
F Fillet · ChamferRound / bevel a corner — two lines, or a polyline corner. Type the radius/distance first; the line under the cursor highlights blue when you can pick it. Click each line on the side you want to keep.
O OffsetClick a polyline on the side you want the parallel copy; type the distance.
Del DeleteRemoves the selection. Ctrl+Z / Ctrl+Y undo / redo.

Layers

The panel (Shift+L) is your inspector. New geometry lands on the active layer (click a name to set it). Per layer: a visibility eye, a colour swatch (click to pick from a palette or a custom colour), an opacity slider, and a lock. Right-click a layer for Isolate / Show All / Lock / Rename / Delete; drag the ⠿ grip to reorder the draw z-order. Deleting a layer moves its geometry to layer 0 (nothing is lost).

Blocks — reusable symbols

  • Make Block — select geometry → name it → pick a base point. It becomes a reusable definition, and the selection turns into an instance in place.
  • Insert (I) — place as many instances as you like.
  • Instances are live: edit the definition and every instance updates (define once, place many). They move / rotate / scale / array / delete as one unit and round-trip to DXF.

Attributes — data on your symbols

Attributes turn a symbol into a data record (a drillhole collar that carries its hole-ID).

  • Define Attribute — place a tag (e.g. HOLEID); include it when you Make Block.
  • Insert then prompts for each value; the value renders as a label that follows the instance.
  • Double-click an instance to edit its values (empty Enter keeps the current one).

Measure

The Measure tool reads distance, azimuth (0° = North, clockwise — geological), and Δx/Δy between snapped points; chains point-to-point.

Files & data

  • Open / Save DXF — Ctrl+O / Ctrl+S (or drag a .dxf onto the board). True arcs, blocks, attributes and layers all round-trip.
  • Export Features (WKT/CSV) — writes the selection (or whole drawing) as a feature table: a wkt geometry column + attribute columns. Block instances export as points carrying their attributes — drops straight into QGIS / GeoPandas / PostGIS. The CRS rides along as an EWKT SRID.
